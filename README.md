## 서비스 소개

핍스는 반려동물을 사랑하는 사용자들을 위해 편의성을 제공하는 서비스입니다. 

1. 맵 기능을 통해 주변 동물병원, 공원을 쉽게 찾아볼 수 있습니다.
2. 장소별로 리뷰를 조회하거나 등록할 수 있습니다. 
3. 반려동물과 관련하여 다양한 정보와 경험을 공유할 수 있는 게시판을 제공합니다.
